Reset Election (Very Important)

Blockchain data cannot be deleted.
So how do we reset?

We use versioning.

Each voter and candidate is linked to an election version.

When reset happens:

electionVersion increases

Old voters/candidates become invalid

System behaves like a fresh election

Old data still exists, but contract ignores it.
